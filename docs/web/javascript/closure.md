# JavaScript Closure: The Beginner's Friendly Guide 

感谢闭包回调，事件处理，高阶函数可以访问到外部作用域的变量。闭包概念在函数式编程中很重要而且经常会在 JavaScript 面试中被问到。

尽管闭包在各个地方都被使用，但它们很难理解。如果你还没有领悟到闭包的“啊哈！”时刻，那么这篇文章就是为你准备的。

## Scope 作用域

作用域由函数或者代码块创建。

不同的作用域可以有相同名称的变量。
